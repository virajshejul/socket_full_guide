What is a Socket?

A socket is an endpoint that allows two devices to exchange data. It acts like a bridge connecting applications across a network.

Types of Sockets

  • Stream Sockets (TCP): Reliable, connection-based.
  • Datagram Sockets (UDP): Fast, no connection required.
  • Raw Sockets: Direct access to network layers.
  • WebSockets: Real-time browser communication.

Socket Protocols

TCP ensures data integrity, while UDP focuses on speed. Both are widely used across different applications.

Common Functions

  • socket() ‚Äì Create a socket.
  • bind() ‚Äì Assign IP & port.
  • listen() ‚Äì Wait for clients.
  • accept() ‚Äì Accept connections.
  • connect() ‚Äì Connect to server.
  • send()/recv() ‚Äì Data exchange.
  • close() ‚Äì End session.

Client-Server Flow

The client initiates connection, and the server responds. Together they exchange packets using the chosen protocol.

Code Examples

Python Server

import socket
server = socket.socket()
server.bind(('localhost', 8080))
server.listen(1)
print("Server listening...")
conn, addr = server.accept()
print("Connected to", addr)
conn.send(b"Hello from Server")
conn.close()

Node.js Example

const net = require('net');
const server = net.createServer(socket => {
  console.log("Client connected");
  socket.write("Hello Client!");
});
server.listen(8080);

Security

  • Use SSL/TLS encryption.
  • Validate user input & sanitize data.
  • Implement authentication & token-based access.

Concurrency

To handle multiple clients:

  • Thread-based: Each connection uses one thread.
  • Async model: Non-blocking (used in Node.js, asyncio).

Error Handling

  • ConnectionRefusedError: Server not available.
  • TimeoutError: No response within limit.
  • BrokenPipeError: Unexpected disconnect.

Performance Optimization

  • Use caching & compression.
  • Reduce latency using UDP for fast apps.
  • Optimize socket buffer size.

Real-world Uses

  • Chat apps (WhatsApp, Discord)
  • Games (Multiplayer servers)
  • Video streaming (Netflix, YouTube)
  • IoT communication systems
